Да, вместо метода `.sendFile()` вы можете использовать `.send()`, 
но это требует немного другого подхода, особенно если 
вы хотите отправить HTML-файл. Вот в чем разница между ними и как их использовать:

1. **sendFile**:
   - Этот метод специально предназначен для отправки файлов клиенту. 
     Он автоматически устанавливает заголовки `Content-Type` 
     в соответствии с типом файла, который вы отправляете.
   - Пример:
     ```javascript
     res.sendFile(path.join(pathTemplate, "index.html"));
     ```

2. **send**:
   - Метод `.send()` используется для отправки содержимого, 
     включая строки, объекты и массивы. Если вы хотите отправить файл, 
     вам нужно сначала прочитать файл и передать его содержимое в `send()`.
   - Пример:
     ```javascript
     fs.readFile(path.join(pathTemplate, "index.html"), (err, data) => {
       if (err) {
         return res.status(500).send("Ошибка загрузки файла.");
       }
       res.send(data); // data - это содержимое файла
     });
     ```

Таким образом, использование `.sendFile()` более удобно и предпочтительно, 
если вы хотите просто отправить файл клиенту, так как 
это делает код более чистым и простым. Метод `.send()` 
следует использовать для отправки строк или объектов 
или для более сложных случаев, когда необходимо 
модифицировать содержимое перед отправкой.
